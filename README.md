# Roadmap Tracker

Created a tracker using Claude Code, to keep track of my own [roadmap](https://github.com/novisiogg/notes)

## Features

- **Global Progress Tracking** –  Completion percentage
- **Phase Organization** – Each phase contains a theory section and a projects section
- **Interactive Checkboxes** – Check off completed tasks, progress updates automatically
- **Phase-level Progress Bars** – Visual feedback for each phase
- **Search & Filter** – Find specific tasks or phases fast
- **Dark / Light Mode** – Toggle between dark & light mode
